# Kategori-Sulit-Soal-2
1.Program meminta ukuran matriks (baris dan kolom).
2.Menggunakan array 2D (int[][]) untuk menyimpan data dari dua matriks.
3.Pengguna mengisi elemen matriks pertama dan kedua.
4.Elemen dijumlahkan satu per satu dan disimpan dalam matriks hasil.
5.Matriks hasil ditampilkan dalam bentuk tabel.
